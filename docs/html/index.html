GEMINI is a software package that solves problems involving blood flow to the brain in the event of acute ischaemic stroke in a patient.

This is part of the EU GEMINI project which seeks to create digital twins of stroke patients. A subset of this project is INSIST (IN-Silico trials for acute Ischaemic STroke), which seeks to model real-world clinical trials of the same.

Note

This documentation is for the perfusion model of the GEMINI software.

GEMINI: https://dth-gemini.eu/

INSIST: https://insist-h2020.eu/

GitHub: https://github.com/Gemini-DTH/perfusion_and_tissue_damage
